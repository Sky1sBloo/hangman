# Hangman
Basic Hangman Game using C#

## Requirements
1. Dotnet

## Installation
### Linux / macOS
1. Run the project
```
dotnet run
```

### Windows
1. Open `Hangman.sln` in Visual studio **or** run:
```
dotnet run
```

## How to play
Input letters that you guess is the correct answer. You have 6 lives.
